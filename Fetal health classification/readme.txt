The folder has 3 code files that can be run.
1. fh_dt.py : python fh_dt.py to run decision tree,bagging and boosting
2. nn.py : python nn.py to run neural network classifier
3. svm.py: python svm.py to run SVM

Methods/classes are imported from following files:
decision_tree.py
bb.py
neural.py
utils.py
